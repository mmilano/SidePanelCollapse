It turns out that there is a hook for horizontal collapse. While height collapse is the default behavior, the bootstrap code will check if the collapsing element has a 'width' class. ```javascript _proto._getDimension = function _getDimension() { var hasWidth = $(this._element).hasClass(Dimension.WIDTH); return hasWidth ?
