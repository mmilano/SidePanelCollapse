Bootstrap v4-based Collapsing Sidebar

{{#heading-L2}} A simple example {{/heading-L2}}

This is a simple single page example, with a set of links in the side navigation that conceptually lead to other pages (although here they just lead back to this page).

It demonstrates the most simple and most minimal way of using the SidePanelCollapse library on a web page.

  • Inclusion
    • The precompiled SidePanelCollapse javascript and css files are linked in the HTML and loaded independently (and asynchronously, in the case of the .js) of the other site resources.
  • Initialization
    • The sidepanel is initialized and created 'automatically' based on the presence of the data-sidepanel-collapse attribute.
  • Configuration
    • Only the default SidePanelCollapse configuration settings are used for initialization.
