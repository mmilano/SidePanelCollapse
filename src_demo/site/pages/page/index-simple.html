Bootstrap v4-based Collapsing Sidebar

{{#heading-L2}} A simple example {{/heading-L2}}

This is a simple single page example, with a fictitious set of links in the side navigation.

It is designed to be about as simple as possible, and to demonstrate the most minimal way of using the SidePanelCollapse library in a site.

  • Library Inclusion
    • The precompiled core SidePanelCollapse javascript and css styles are linked in the HTML and loaded independently of the other site scripts and styles.
  • Configuration
    • The default SidePanelCollapse configuration settings are used for initialization.
