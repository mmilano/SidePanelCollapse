The sidebar navigation is not being displayed right now because the window width is too wide.

Resize the window to a smaller width in order to display the sidepanel navigation.

This is an example comprising a set of multiple pages that have a side navigation. As a site, these pages demonstrate more complex ways of using the SidePanelCollapse library (compared to the Simple Demo).

  • SidePanel Library Inclusion
    • The SidePanelCollapse javascript is incorporated into the single site.js file during the build process
    • The SidePanelCollapse css styles are incorporated into the single site.css file during the build process
  • SidePanel Configuration
    • Custom SidePanelCollapse configuration settings are used, and are defined differently between the index page and the interior pages (i.e. the other pages).

On this page, the site's index page, the primary navigation toggles responsively to the window width.
