{{#heading-L2}} About this demo {{/heading-L2}}

This is an example set with multiple pages that have side navigation. As a site, these pages demonstrate more complex ways of using the SidePanelCollapse library in a project (as compared to the Simple Demo).

  • Inclusion
    • The SidePanelCollapse javascript source is incorporated into the single site.js file during the build process that uses Browserify and Babel.
    • The SidePanelCollapse SCSS source is incorporated into the build process and compiled as part of the single site.css file.
  • Initialization
    • The sidepanel is initialized programmatically in javascript by a script specific to each page.
  • Configuration
    • Custom SidePanelCollapse configuration settings are used.
    • The index page uses different settings than the interior pages.
